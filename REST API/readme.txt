1. Install Python and Visual Studio Code setups
2. Open the REST API folder in VS Code
3. Install python extension
4. Open terminal
5. Create virtual environment by entering the following command
	python -m venv .venv
6. Select the venv interpreter by ctrl+shift+p
7. Reopen the terminal
8. Install requirements.txt by entering the following command
	pip install -r requirements.txt
9. Make sure the path to fruits_model.h5 is correct
10. Run the server by ctrl+f5
